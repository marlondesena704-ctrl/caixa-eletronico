# 🦇 Caixa Eletrônico Wayne-Trust

Simulador de caixa eletrônico bancário, rodando 100% no navegador (HTML + JavaScript, sem backend).

Projeto de estudo em Java, adaptado para uma versão web interativa, com:

- Login por usuário e senha
- Consulta de saldo (BAT-DÓLARES)
- Saque com validação de saldo insuficiente e valor negativo
- Depósito
- Cálculo do menor número de notas entregues no saque (algoritmo guloso), usando notas de 200, 100, 50, 20, 10, 5 e 2
